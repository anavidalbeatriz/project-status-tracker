# Goal
The project's goal is to make an application that will have all of the company's projects and will alow tech leads to update about the project status. This update will be made by a transcription of a meeting passed trough the AI, that will format the information from the transcription and update the project status.

# Project status
There are 7 main subjects that should be updated by the AI using the transcription:
- Client
- Project
- Is the project on scope?
- Is the project on time?
- Is the project on budget?
- What is the next delivery?
- What are the project risks?
The AI should use the transcription to update these 5 topics and this update will be reflected in the system

# The application
The application should be capable of:
- Inserting new project
- Exclude projects
- Edit project
- Populate each project with the information from project status
- Add users
- Produce a report with the projects health
